### What is API?
- API stands for Application programming interface.
- It is one of the way used by one software program/application to communicate with another software program/application for data exchange.
  - Other communication types are FTP. 
- API follows request-response model. Client makes request and server returns response.
- Server prepares the API contract and client follows that contract to make a call.
- API defines set of rules to talk with server. For example rules like
  - API endpoint name.
  - HTTP Method name.
  - Protocol name
  - Request body format
  - headers, query parameters and uri parameters required.
- If a web application or mobile application wants to talk with another application over internet they use API.
- API defines, pre determined ways for a software application to interact with other software application.

### What is API Endpoint?
* API endpoint is unique or specific URL, used by client to make request to server to get response back.
