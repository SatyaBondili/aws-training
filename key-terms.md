### What is Cloud Computing?
* Renting computer and resources like storage, database, servers and network over internet is called Cloud Computing.
* In Cloud computing, cloud means getting anyting over internet is called Cloud.
* Delivery of IT resources over internet on demand, based on pay as you go model is called Cloud Computing (AWS Defination)
  
**Computing means** processing information using computers.

### Difference between AWS and AWS partner network?
* AWS provides only infrastructure.
* AWS partner network is AWS approved companies whose main objective is to help customers who want to use AWS cloud services.
  - AWS don't provide support directly to its customers, because they get less margin profits with more labour maintenace headache. Hence it builds only infrastructure and services.
  - A Company which is part of AWS partner network subscription, helps to prepare solution design for customers to migrate to AWS infrastructure. AWS parnter network company also assists production issues troubleshooting.
